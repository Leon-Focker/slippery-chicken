Please Use GitHub

Since 2017 slippery chicken uses github to manage its code repository. From 2020, due to increasing problems relating to Mac OSX apps, slippery-chicken will only be available as a source code download. The information and instructions below can be considered deprecated. See the github wiki entry for how to install slippery chicken and using github for more information.

Software downloads

Below, you'll find the source code, OSX app, and pre-compiled Lisp images for slippery chicken. The sc- files are the full source code and documentation; please follow the installation instructions to install from source.

Pre-compiled Lisp images

Available up to and including version 1.0.8: The -precompiled files include Clozure Common Lisp and a pre-compiled image of slippery chicken. This is the easiest version to install and is recommended for newcomers to Lisp and slippery chicken who run OSX 64-bit or Windows 7 32-bit. This will allow CMN, Lilypond, and MIDI file writing but no sound file writing with CLM.

OSX application

The .app downloads work on Macintosh OSX only and include CM, CLM, and CMN along with slippery chicken, but to reduce download size, no documentation. The .app requires Emacs and a C compiler (both free) however; see the readme file that comes with the app for more details.

Documentation

The documentation is always available online but if you want a local copy please download the source code below (i.e. sc-*.tar.gz, not the app).

Subversion access

Subversion server access at the University of Edinburgh is no longer available. Source code is however clonable from github.

Installation instructions, bugs, documentation, etc.

Please see the installation instructions and bug report page, as well as the manual for more details.
